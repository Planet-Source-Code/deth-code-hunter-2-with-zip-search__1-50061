Code Hunter v2

The required dll 'unzip32.dll' is stored in the res file, it will auto install when you open code hunter if its not installed. You must have the visual basic 6 runtimes (sp5) installed and the mscomtctl.ocx from microsoft. The download links are below if you need them.

VB6 Runtime:
http://download.microsoft.com/download/vb60pro/Redist/sp5/WIN98Me/EN-US/VBRun60sp5.exe

VB6 Common control:
(you will need to register this probably once installed. Run > regsrv32 mscomctl.ocx)
http://activex.microsoft.com/controls/vb6/MSComCtl.cab
